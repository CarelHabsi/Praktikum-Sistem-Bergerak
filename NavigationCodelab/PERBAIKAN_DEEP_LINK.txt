Kenapa tidak navigasi sebelumnya?
   RallyActivity menerima Intent dengan URI deep link, tapi tidak
   meneruskannya ke NavController. NavController tidak tahu harus
   navigasi ke mana.

Bagaimana solusinya?
   1. Pass Intent dari Activity ke Composable (RallyApp)
   2. Gunakan LaunchedEffect untuk handle deep link saat composable di-load
   3. Extract URI dari Intent dan pass ke navController.navigate(uri)

LaunchedEffect itu apa?
   LaunchedEffect adalah composable yang menjalankan side effect
   (seperti navigation) saat composable pertama kali di-compose.

   LaunchedEffect(intent) { ... }
   - Akan jalan sekali saat composable pertama kali muncul
   - Akan jalan lagi jika value 'intent' berubah

   - Temukan RallyActivity (dari AndroidManifest.xml intent-filter)
   - Start RallyActivity dengan Intent yang berisi URI

3. RallyActivity.onCreate():
   - Terima Intent dengan data URI
   - Pass Intent ke RallyApp composable

4. RallyApp composable:
   - LaunchedEffect berjalan saat pertama kali di-compose
   - Extract URI dari Intent: rally://single_account/Checking
   - Call navController.navigate(uri)

5. NavController:
   - Match URI dengan deepLinks yang terdaftar
   - Temukan composable SingleAccount dengan pattern:
     "rally://single_account/{account_type}"
   - Extract argument: account_type = "Checking"
   - Navigate ke SingleAccountScreen dengan accountType="Checking"

6. SingleAccountScreen:
   - Tampilkan halaman detail account dengan tipe "Checking"
